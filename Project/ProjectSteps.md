# Project Steps

```txt
1) Select Public cloud provider to create kubernetes cluster - AWS ( EKS ), Azure ( AKS ), Google ( GKE ). Create the cluster with 2/3 nodes.
```

```txt
2) Select a application to deploy - Front end, Back end and Database. Create a GitHub Repo and start adding your code and dockerfiles.
```

```txt
3) Containerize the application using Dockerfile. Create Version 1 - DockerImages for different microservices
like front-end, back-end and DB. Push these images to DockerHub or any private container registry like ECR, ACR etc
```

```txt
4) Start writing your manifest files. Deployment, Services, Env, ConfigMap, Secret - Managing and co-ordination
```

```txt
5) Start creating objects and try to load application in Browser
```
